# Student Management System

A simple Java console-based application for managing student records using Object-Oriented Programming (OOP) principles and CRUD operations.

## Features
- View all students
- Search student by ID
- Add new students
- Update student information
- Delete student records
- Menu-driven console interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner
- Git
- GitHub

## Project Structure

```
StudentManagementSystem/
│
├── Student.java
├── StudentManagementSystem.java
└── README.md
```

## CRUD Operations

### Create
Add new students to the system.

### Read
View all students or search by student ID.

### Update
Modify student name and age.

### Delete
Remove students from the system using their ID.
